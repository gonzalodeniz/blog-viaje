# frontend

React 18 + Vite + TypeScript, TanStack Query y editor TipTap. Build estático servido por nginx. Esqueleto generado con `npm create vite@latest -- --template react-ts` (TASK-R1-001); pendiente de completar en WP-R1-5/6 — ver [ADR-001](../specs/adr/ADR-001-stack-tecnologico.md).

## Desarrollo

```bash
npm install
npm run dev       # servidor de desarrollo
npm run build     # build de producción
npm test          # Vitest
```
