# Architecture Decision Records

Decisiones de arquitectura numeradas (`ADR-nnn-titulo.md`), inmutables una vez aceptadas: para revertir o cambiar una decisión se escribe un nuevo ADR que la sustituye (`Estado: sustituida por ADR-xxx`).

Formato: **Contexto** (qué fuerza la decisión) · **Decisión** · **Alternativas descartadas** (con el porqué) · **Consecuencias** (incluidas las negativas).
